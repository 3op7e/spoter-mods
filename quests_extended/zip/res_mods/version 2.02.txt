Название: Потапыч (quests_extended)
Текущая версия 2.02 для 0.9.8 (31/05/2015)

Описание:
Мод Автоматически активирует доступные ЛБЗ. Переактивация происходит в ангаре, при выборе танка в карусели.
На данный момент альфа. с минимальным функционалом.

	Исправлена ошибка, приводившая к зависанию при наличии незабранной награды.

Обновлено в версии 1.02
	Добавлена всплывающая подсказка с описанием ЛБЗ при загрузке боя, до его начала.
	Добавлена интеграция с модом Танкопыт http://forum.worldoftanks.ru/index.php?/topic/1485101- 
	Совместо с модом Танкопыт, показывает информацию о боевых задачах в Ангаре и позволяет на лету переключать режим сложности, для активации ЛБЗ.
	Режим сложности Повышенная: активируются сначала задачи, которые выполнены без Дополнительных требований, для повторного прохождения
	Режим сложности Нормальная: активируются сначала задачи, которые еще не проходились ранее
	Кнопки активации переключения режимов, доступны только совместо с модом Танкопыт, вручную можно поменять в конфиге, опция <Hard_mode>

Обновлено в версии 1.01
	Добавлен режим выбора Тяжелый (Hard mode), автоматически пытается активировать пройденую не до конца задачу(не выполнены дополнительные условия)
	Исправлены ошибки заполнения списка задач при различных условиях

Обновлено в версии 1.00
	Если при выборе танка есть свободные слоты для ЛБЗ, то мод автоматически заполняет их доступными задачами, сначала добавляются те задачи которые доступны для прохождения, минимального уровня, потом все остальные.
	В дальнейшем алгоритм заполнения будет изменен. Это альфа версия.
